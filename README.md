# Fokus

A simple productivity timer web app in the style of Pomodoro technique. It includes focus, short break, and long break modes, background music toggle and start/pause control. Designed with HTML, CSS and vanilla JavaScript.

## Features

- Focus mode (25 minutes)
- Short break mode (5 minutes)
- Long break mode (15 minutes)
- Start/Pause/Resume button with icon toggle
- Optional background music switch
- Finish sound and alert when timer ends

## Project structure

- `index.html` - page structure and buttons
- `styles.css` - layout and styling
- `script.js` - timer logic, DOM manipulation and interaction
- `imagens/` - icon and banner images
- `sons/` - audio files

## Usage

1. Open `index.html` in a browser.
2. Select a mode: Focus, Short Break or Long Break.
3. Click the main button to Start.
4. Click again to Pause; click once more to Resume.
5. Toggle music with the switch.

## Notes

- Paths are configured for local assets: `/imagens/*` and `/sons/*`.
- The app is intended as a simple frontend demo; no backend required.
